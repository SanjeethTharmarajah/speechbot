## 🧠 Features

- 🎙️ **Speech Recognition (Speech-to-Text)** — capture vocal input and convert it to text.
- 🔊 **Text-to-Speech Playback** — read typed text out loud.
- 🎛️ Adjustable settings (e.g., voice selection, language, speed).
- 🎯 Browser-based (no external software required).
- 📱 Works on desktop and mobile browsers that support Web Speech API.

## Requirements
1. Get API Key from groq at https://console.groq.com/keys
2. Add API key in config.php
3. Use browsers like Chrome or Edge which supports Web Speech API (This only works in Web Speech API supported browsers only)
